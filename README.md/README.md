# IPL Ticket Demand Forecasting using Synthetic Data

## 📌 Project Overview
This project analyzes factors affecting IPL match ticket sales and builds a predictive model to estimate audience turnout using synthetic data.

---

## 🎯 Business Problem
Sports event organizers need to estimate ticket demand in advance to optimize scheduling, stadium allocation, and maximize revenue.

---

## 🧠 Methodology
- Synthetic dataset generation  
- Data cleaning and validation  
- Exploratory data analysis  
- Data visualization  
- Linear regression modeling  

---

## 📊 Key Insights
- Weekend matches tend to attract higher attendance  
- Popular teams significantly increase ticket sales  
- Larger stadiums support higher ticket sales  
- Weather impact is less consistent compared to other factors  

---

## 🤖 Model Prediction
The trained model predicts approximately **50,850 ticket sales** for a high-demand weekend match with favorable conditions.

---

## ⚠️ Limitations
- Dataset is synthetically generated  
- Results may not reflect real IPL conditions  

---

## 🚀 Future Improvements
